# Cofrinho de Moedas 🐖

Esse é um projeto feito para a disciplina de Programação Orientada a Objetos da UNINTER, onde foi implementado, na linguagem Java, um cofrinho que o usuário pode adicionar moedas de Real, Euro e Dólar.
